author: 朱宇浩 <dr.yuhao.zhu@outlook.com>

本方案适合 iRime。
部署方法：
    将本文件夹中两个文件拷贝到 build 文件夹下，
        - clover_zrm_14.prism.bin
        - clover_zrm_14.schema.yaml
    同时在根目录下的 default.custom.yaml 中的 schema_list 添加本方案名 "- schema: clover_zrm_14"。
    重新部署后即可使用。

description:
🍀️四叶草自然码双拼14键，是一款适合在手机上使用的方案。
    <https://github.com/forFudan/rime/xuma>
    它基于以下目的设计：
    - 使用🍀️四叶草简体拼音的词库
    - 使用自然码双拼方案
    - 使用14键布局
    - 输入编码实时反应为第一候选字词的编码

配合 cosmic_14key 皮肤使用效果更佳。

制定本方案时，参考了以下方案的设计：
